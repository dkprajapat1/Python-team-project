# 🚀 Python Team Project

This repository contains our team-based project developed for college coursework. The project is built collaboratively and includes a full-stack setup with a backend server, a React frontend, and an AI service.

---

## 📁 Project Structure

* **backend/** → Node.js backend server
* **react-frontend/** → React-based frontend
* **ai-service/** → Python AI service

---

## ⚙️ Setup & Run Instructions

Follow the steps below to run the complete project:

---

### 🖥️ 1. Start Backend Server

```bash
cd backend
nodemon index.js
```

---

### 🌐 2. Start Frontend

Open a new terminal:

```bash
cd react-frontend
npm install
npm run dev
```

---

### 🤖 3. Start AI Service

Open another terminal:

```bash
cd ai-service
venv\Scripts\activate
python app.py
```

You should see:

```
Starting server on port 8000...
```

---

## 🔗 Access the Application

* Frontend → http://localhost:3000 (or as shown in terminal)
* Backend → http://localhost:5000 (if configured)
* AI Service → http://localhost:8000
